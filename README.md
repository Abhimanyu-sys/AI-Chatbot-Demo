#  AI Chatbot Demo

# Overview

An AI-powered chatbot application built using Python and web technologies.
This project demonstrates how conversational AI can be integrated into real-world applications.

---

# Features

* 💬 Interactive chatbot interface
* 🤖 AI-based response generation
* 📂 Data handling using CSV
* 🌐 Web-based UI
* ⚡ Fast and lightweight implementation

---

# Tech Stack

### Frontend

* HTML
* CSS

### Backend

* Python

### Data Handling

* CSV (leads.csv)

---

# Project Structure

```
AI-Chatbot-Demo/
│
├── Index.HTML        # Frontend UI
├── app.py            # Backend logic
├── leads.csv         # Data file
├── requirements.txt  # Dependencies
```

---

# Setup Instructions

# Clone Repository

```bash
git clone https://github.com/your-username/AI-Chatbot-Demo.git
cd AI-Chatbot-Demo
```

# Install Dependencies

```bash
pip install -r requirements.txt
```

# Run Application

```bash
python app.py
```

---

# How It Works

* User interacts with chatbot via web interface
* Backend processes input using Python logic
* Data is fetched/processed using CSV file
* Response is generated and returned
